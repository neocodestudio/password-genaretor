Overview

SecureGen ("the App", "we", "our") is a password generation and management application built for Android using Flutter. This Privacy Policy describes what information the App interacts with, how it is handled, and the security measures in place to protect you.

SecureGen is designed with a privacy-by-design philosophy. The App does not operate a backend server, does not connect to any cloud service, and does not transmit your data anywhere. Everything stays on your device.

Our Core Promise

SecureGen will never collect, transmit, sell, share, or store your personal data on any external server. Your passwords are yours ‚Äî and only yours. This is a technical guarantee, not just a policy statement.

Internet Requirement

SecureGen requires an active internet connection (WiFi or Mobile Data) to operate. This is a product-level requirement ‚Äî not because data is sent to a server, but as a built-in product design decision.

‚úÖ
Verified: No Data Is Transmitted

Even though an internet connection is required, SecureGen does not send, upload, sync, or transmit any data to any server ‚Äî ever. You can verify this by using a network traffic monitor (e.g., Wireshark or Android's Network Inspector) while using the App ‚Äî zero outbound data will be detected.

The App uses the connectivity_plus Flutter package solely to detect whether a network interface is active. No network requests are made through this package or any other mechanism within the App.
